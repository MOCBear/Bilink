# Bilink 个人交互主页系统

Bilink 是一款采用现代化液态玻璃视觉效果的个人交互主页系统，支持前后端分离架构，提供美观且功能完整的个人展示平台，包含访客浏览和管理员管理双重角色权限。注意吼：只是一个摸鱼写的项目，存在诸多BUG等你去发现。

## 核心特性

- **视觉设计**：集成 iOS 26 风格的液态玻璃（Glassmorphism）效果，扁平化现代设计
- **功能模块**：支持个人信息、技能、项目、社交链接展示与管理
- **权限管理**：区分访客 / 管理员角色，JWT 认证保障管理后台安全
- **响应式布局**：适配手机、平板、桌面等多设备尺寸
- **便捷管理**：管理员可通过控制面板一编辑所有展示内容（三击底部版权信息将会得到登录按钮）
- **离线图标库**：内置丰富的离线字符图标库，支持技能图标自动匹配
- **图标选择器**：提供直观的图标选择界面，支持图标搜索和预览

## 技术栈

### 前端

- React 18 + React Router DOM 6
- Vite 6 构建工具
- Axios 网络请求
- CSS3 液态玻璃样式实现

### 后端

- Node.js 24 + Express 4
- JWT 身份认证
- bcryptjs 密码加密
- SQLite 数据库存储（用户信息）
- JSON 文件存储（个人资料数据）

## 快速开始

### 环境要求

- Node.js ^24.11.1
- npm ^10.0.0

### 安装部署

```bash
# 克隆项目
git clone https://github.com/MOCBear/bilink.git
cd bilink

# 安装后端依赖
cd backend
npm install
cp .env.example .env

# 安装前端依赖
cd ../frontend
npm install

# 启动后端（开发模式）
cd ../backend
npm run dev

# 启动前端（新终端）
cd frontend
npm run dev
```

### 访问应用

- 前端页面：http://localhost:3000 或 http://localhost:3001（如果3000端口被占用）
- 后端 API：http://localhost:5000

## 使用指南

### 访客使用

直接访问主页即可浏览个人信息、技能、项目等内容，点击社交链接可跳转至对应平台。

### 管理员登录

1. 在主页任意位置点击 3 次触发登录按钮
2. 使用默认账号（admin/admin123）登录管理后台
3. 登录后可编辑个人信息、技能、项目等所有内容
4. 首次登录建议修改默认密码保障安全

### 管理员控制面板使用

#### 基本信息管理

- **基本信息**：编辑个人姓名、头衔、头像、简短介绍和详细介绍
- **技能专长**：添加、编辑、删除技能，支持技能图标自动匹配和手动选择
- **项目作品**：添加、编辑、删除项目，包含项目名称、描述、链接、图片和标签
- **社交链接**：添加、编辑、删除社交账号链接，支持多种平台类型
- **联系方式**：编辑电子邮箱和所在地信息
- **账户管理**：修改用户名和密码

#### 技能图标使用

1. **自动匹配**：输入技能名称时，系统会自动匹配对应的图标（如输入 "JavaScript" 自动匹配 "🟨"）
2. **手动选择**：点击图标输入框右侧的 🎨 按钮打开图标选择器
3. **图标搜索**：在图标选择器中输入关键字搜索图标
4. **图标预览**：鼠标悬停在图标上可以预览效果
5. **图标应用**：点击图标即可应用到技能项

### 账户管理

1. **修改用户名**：在账户管理页面修改用户名，点击 "更新账户信息" 按钮保存
2. **修改密码**：输入当前密码和新密码，系统会验证密码复杂度，点击 "更新密码" 按钮保存
3. **密码要求**：密码长度不少于8位，包含大写字母、小写字母和特殊符号

## 生产部署

1. 前端构建：`cd frontend && npm run build`，将 dist 目录部署至静态服务器
2. 后端部署：
   - 配置环境变量（修改 JWT_SECRET）
   - 使用 PM2 守护进程：`pm2 start server.js --name bilink-backend`
3. 数据库备份：定期备份 `backend/data/bilink.db` 文件
4. 生产环境建议配置 CORS 白名单，限制前端访问域名

## 常见问题

- **登录按钮不显示**：确保主页点击满 3 次，检查浏览器控制台无 JS 错误
- **液态玻璃效果异常**：使用 Chrome/Firefox/Edge 现代浏览器（需支持 backdrop-filter）
- **忘记管理员密码**：删除 `backend/data/bilink.db` 后重启后端，恢复默认密码
- **端口被占用**：修改 `backend/.env` 中的 PORT 或前端 `vite.config.js` 端口配置
- **图标选择器不显示**：确保浏览器支持弹出窗口，检查浏览器控制台无 JS 错误

## 数据存储说明

### 用户信息

- 存储位置：`backend/data/bilink.db`（SQLite 数据库）
- 包含内容：用户名、密码（加密存储）、角色
- 安全机制：密码使用 bcryptjs 加密存储

### 个人资料

- 存储位置：`backend/data/data.json`（JSON 文件）
- 包含内容：个人信息、技能、项目、社交链接、联系方式
